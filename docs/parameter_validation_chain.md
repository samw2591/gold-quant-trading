## 核心原则

1. **所有参数变更都经过组合验证** — 不是从独立实验中拼凑，而是在前一步组合基础上增量测试
2. **所有回测包含交易成本** — $0.50 点差是标准测试条件
3. **K-Fold 交叉验证是上线门槛** — 至少 4/6 折优于基准才实装
4. **PBO (回测过拟合概率) = 0.00** — 通过 CSCV 70 种组合验证，零过拟合

## 参数 1: 追踪止盈 — Trail Activate 0.8 / Distance 0.25

| 项目 | 详情 |
|---|---|
| **之前** | Activate 2.5 ATR / Distance 0.5 ATR |
| **之后** | Activate 0.8 ATR / Distance 0.25 ATR |
| **质疑** | "浮盈很小就激活，容易被正常回撤震出" |

### 为什么旧值是错的

- 2.5 ATR ≈ $110 的浮盈才激活追踪止盈
- 在日内 H1 级别交易中，单笔浮盈超过 $110 极为罕见
- **追踪止盈在旧参数下形同虚设** — 几乎从未触发

### 验证证据

| 阶段 | 范围 | 结果 |
|---|---|---|
| R1 (7 变体) | Trail Act 1.0-2.5 | Act 1.5 最优: Sharpe +0.24, MaxDD -26% |
| R2 (22 变体) | Act 0.6-1.2 × Dist 0.15-0.40 | 0.8/0.25 最优: Sharpe 0.39→1.74 (+346%) |
| 过拟合检测 | ±1 步参数扰动 | 所有扰动 Sharpe > 2.0, Act 0.6/Dist 0.20 甚至更好 |
| R3 组合 | 与 SL/TP/ADX 叠加 | 组合 Sharpe 2.54, 11 年逐年全正 |
| K-Fold 6 折 | C12 (含 Trail) | 5/6 折优于基准 |
| 带成本验证 | $0.50 spread | Sharpe 1.03 (C12+Adaptive), 仍然核心 alpha |

### "容易被震出"的反驳

- 追踪止盈触发 5,542 笔, 胜率 **97.7%**, 总利润 +$41,088 — 这是**整个策略的核心利润引擎**
- 旧参数 (2.5/0.5) 下追踪几乎不触发, 利润全部靠 timeout/SL, Sharpe 仅 0.39
- 赢家中位持仓 6 bars (1.5h), 快进快出; 输家中位 46 bars (11.5h), 拖很久 — 早激活追踪恰恰符合这个特征

## 参数 2: 止损 — SL 4.5 ATR

| 项目 | 详情 |
|---|---|
| **之前** | 2.5 ATR |
| **之后** | 4.5 ATR |
| **质疑** | "每笔亏损从 ~$50 变成 ~$90" |

### 为什么放宽止损反而更好

- **Keltner 是突破策略**: 价格突破通道后常回测通道再继续趋势
- 2.5 ATR 止损在回测阶段频繁被震出, 然后价格继续原方向走
- SL 放宽 → 减少假突破被震出 → 胜率提高 → 弥补单笔额度

### 验证证据

| 阶段 | 结果 |
|---|---|
| R3 组合 | SL 3.5 vs 2.5: Sharpe +0.49, 组合内验证 |
| Mega Grid (1440 组合) | SL 4.0-4.5 区域是全局最优带 |
| SL 精调 (带 $0.50 成本) | SL 4.5: Sharpe **1.35**, MaxDD **$559** — 全维度优于 SL 3.5 (1.03, $732) |
| 逐年验证 | SL 4.5 在 **12/12 年全正** |
| 手数影响 | 手数从 ~0.02 降到 0.01, 但 MaxDD 反而**下降** $173 |

### "亏损翻倍"的反驳

- 名义止损距离翻倍, 但手数自动缩减 (ATR 调仓)
- **实际风险 per trade 不变** — 每笔风险仍是 $50 (2.5% 本金)
- SL 放宽后**止损次数减少**, 总亏损金额反而更少: MaxDD $732→$559

## 参数 3: 止盈 — TP 8.0 ATR

| 项目 | 详情 |
|---|---|
| **之前** | 3.0 ATR |
| **之后** | 8.0 ATR |
| **质疑** | "等于取消了 TP" |

### 这不是 bug, 是 feature

- **TP 本来就几乎不触发**: TP 5.0 仅 1.4% 触发率 (EXP06)
- 真正出场机制是 **trailing stop** (97.4% WR)
- TP 设小 = 人为限制利润上限, 在大趋势日割肉
- TP 设大 = 给利润更多空间, 让 trailing 自然跟踪

### 验证证据

| TP 值 | Sharpe | 触发率 | 结论 |
|---|---|---|---|
| 2.0 ATR | 1.13 | ~5% | 过早截断利润 |
| 5.0 ATR | 1.52 | 1.4% | 基准 |
| 8.0 ATR | **1.56** | <0.5% | 微幅改善, 安全改动 |

- TP 从 5.0→8.0 Sharpe +0.04, 风险零 — **最安全的改动之一**

- 今天 (04-06) 的实盘已验证: ADX=24.9 但 trend_score=0.26 → 系统拒绝开仓
- ADX 18→24 的增量仅 +0.09 Sharpe, 不是关键参数

## 参数 5: KC 通道 — EMA25 / 乘数 1.2

| 项目 | 详情 |
|---|---|
| **之前** | EMA20 / 1.5× ATR |
| **之后** | EMA25 / 1.2× ATR |
| **质疑** | "通道变窄 → 信号更多 → 噪音" |

### 为什么更窄通道不等于更多噪音

- 通道变窄 (1.5→1.2), 但中轨 EMA 更长 (20→25) → **更平滑的中轨减少噪音**
- 净效果: 更敏感捕捉真实突破, 同时过滤短期抖动
- 参数探索 (Test 4): KC 越窄 Sharpe 单调递增 (1.0 > 1.25 > 1.5 > 1.75 > 2.0)

### 验证证据

| 阶段 | 结果 |
|---|---|
| EXP13 (14 变体) | KC25/M1.2 冠军: Sharpe **1.84** (+0.32) |
| EXP20 K-Fold | KC25/M1.2: **4/6 折赢**, Avg Sharpe 0.86→1.17 (+36%) |
| 参数悬崖检测 | KC mult 敏感度 range=0.38, **平滑无悬崖** |
| KC EMA 敏感度 | range=0.44, 平滑无悬崖 |

### EMA150 对比 (被否决)

- EMA150 趋势过滤单独测试很好, 但 K-Fold Fold4 (2021-2022) 崩溃至 5 笔 Sharpe=-13.56
- **这就是为什么我们用 K-Fold 把关** — 独立测试好看不等于组合稳定

## 参数 6: Cooldown — 30 分钟

| 项目 | 详情 |
|---|---|
| **之前** | 3 小时 (3 bars) |
| **之后** | 30 分钟 |
| **质疑** | "很快又能入场, 可能连续亏损" |

### 验证证据

| 点差 | CD=30min Sharpe | CD=3h Sharpe | 差值 |
|---|---|---|---|
| $0.30 | **1.93** | 1.79 | +0.14 |
| $0.50 | **1.35** | 1.21 | +0.14 |
| $0.80 | -0.30 | -0.44 | +0.14 |

- **30min 在所有点差下均优于 3h**, 且改善幅度完全一致 (+0.14)
- 3h 冷却过长, 趋势行情止损后常有延续, 3h 后趋势已经结束
- 冷却粒度扫描: 15-20min (1根M15 bar) 等效, 30-40min (2根M15 bar) 等效

### "连续亏损"的反驳

- 日内亏损上限 5 笔, **硬性保护**
- 亏损递减手数: 1笔后→0.03, 2笔后→0.02, 3笔后→0.01
- Adaptive Trend Gating: choppy 下不开仓, 实质上减少了低质量再入场
